# Security Policy

## 支持版本

TaCOS 目前只对最新公开版本提供安全维护。

| Version | Supported |
| ------- | --------- |
| 2.2.x   | Yes       |
| <= 2.1  | No        |

## 报告漏洞

请不要通过公开 issue 报告安全漏洞。

发布到 `thu-chorus/tacos` 后，请优先使用 GitHub 的 **Report a
vulnerability** 私密报告入口提交漏洞信息。仓库维护者应在公开前启用
GitHub private vulnerability reporting。

报告时请尽量包含：

- 受影响的版本或提交
- 可复现步骤
- 影响范围
- 已知的临时缓解方式

维护者会尽力在 7 天内确认收到报告，并在完成修复或缓解前避免公开披露细节。
